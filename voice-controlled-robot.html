Overview

This project aimed to minimize human intervention for mobility devices. [cite_start]I engineered a system that allows users to control a wheelchair using simple voice commands (Forward, Backward, Left, Right, Stop) via an Android application[cite: 152, 201].

Technical Implementation

1. Power Management

[cite_start]A critical challenge was powering 12V DC Geared Motors (60 RPM) while maintaining torque for an 80kg load capacity[cite: 317]. [cite_start]I implemented a Buck Converter (DC-to-DC) to step down voltage and step up current, ensuring efficiency losses were restricted to 5-15%[cite: 331, 333].

2. Hardware Interface

The system uses an Arduino Uno as the central controller. [cite_start]The HC-05 Bluetooth Module handles data transmission from the phone within a 100m range [cite: 161][cite_start], while the L293D Motor Driver acts as the interface between the low-voltage microcontroller and the high-power motors[cite: 222].

3. Software Logic

[cite_start]The Android app uses Google's speech recognition API to convert voice to text[cite: 165]. [cite_start]This text is sent via Bluetooth to the Arduino, which executes specific functions (e.g., `digitalWrite` High/Low) to control wheel rotation direction[cite: 289].
