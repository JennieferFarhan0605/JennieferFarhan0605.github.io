Voice Controlled Robot
(Wheelchair Implementation)

A self-sufficient robotic vehicle using speech recognition to perform navigation tasks, [cite_start]engineered to assist individuals with disabilities[cite: 51, 56].

Arduino Uno C++ MIT App Inventor HC-05 Bluetooth Buck Converter

Overview

The primary objective of this project was to design a system that minimizes human intervention in mobility devices. By utilizing an Android-based voice recognition application, we successfully controlled a robotic [cite_start]wheelchair via Bluetooth connectivity[cite: 53, 54].

The system processes voice commands (Forward, Backward, Left, Right, Stop) and translates them into [cite_start]motor signals driven by an Arduino microcontroller[cite: 308].

Technical Architecture

1. Hardware Interface

[cite_start]The core is the Arduino Uno (ATmega328), which acts as the brain of the robot[cite: 162]. To handle wireless communication, we integrated the HC-05 Bluetooth Module, operating within a [cite_start]100m range with a data transfer speed of 3Mbps[cite: 275].

2. Power Management (Buck Converter)

A critical challenge was powering the 12V DC Geared Motors (60 RPM) while maintaining sufficient torque for a load [cite_start]of 80kg[cite: 315, 317].

We implemented a Buck Converter (DC-to-DC) to step down voltage and step up current. This ensured [cite_start]high efficiency (losses restricted to 5-15%) and stable power delivery under varying load conditions[cite: 331, 333].

3. Android Integration

The user interface was built using MIT App Inventor. The app handles Speech-to-Text conversion [cite_start]utilizing Google's speech recognition API, then transmits the text string via Bluetooth to the Arduino[cite: 256].
